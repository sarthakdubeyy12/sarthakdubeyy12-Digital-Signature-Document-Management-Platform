🔄 AUTHENTICATION FLOW

1. Registration/Login
   User → POST /auth/register or /auth/login
   Server → Returns accessToken + refreshToken

2. Accessing Protected Routes
   Client → Sends: Authorization: Bearer {accessToken}
   Server → Validates token → Returns data

3. Token Expiry
   Client → Access token expires (15 min)
   Client → POST /auth/refresh with refreshToken
   Server → Returns new accessToken + refreshToken

4. Logout
   Client → POST /auth/logout with refreshToken
   Server → Revokes refresh token
